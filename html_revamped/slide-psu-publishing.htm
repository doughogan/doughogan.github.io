Publishing to the PSU Web Space

First, you need to apply for your PSU web space. You may have to wait up to a week to get a confirmation that your space has been created.

The space you have is on a server that's connected to the Internet. You'll store all the files you want to be your webpage on the server.

There are a few ways to access your space. We'll go over two here:

Getting to your space via the web anywhere

You can get to your space anywhere via the Penn State Portal, as follows:

  1. Open a web browser and go to https://portal.psu.edu/.
  2. Click Login in the top left.
  3. Enter your PSU username and password.
  4. Look for the section titled Penn State Access Account Storage Space. It's in the middle in the defualt layout (I think, unless I configured it...).
  5. Click on the link labeled PASS Explorer.

At this point, you'll see a web interface for doing file management operations and all of the files in your web space.

 

Getting to your space in ITS Windows labs

You'll treat your web space as if it's another drive, specifically a network drive. You need to tell Windows how to find your space, which is called mapping a network drive:

  1. Right-click the My Computer icon on your desktop.
  2. Select Map Network Drive.
  3. In the pull-down menu labeled Drive, choose W.
  4. In the Folder field, type \\SAMBA.CAC.PSU.EDU\HOMES.
  5. Check Reconnect at Logon.
  6. Click Finish.
  7. You may be asked to enter your username and password. If so, they're the same PSU username and password.

The W drive now exists as an option in My Computer. You can use it just like any other drive (except that deleted files do NOT go to the Recycle Bin from here.)

For problems and anything not covered here, see the Directions from ITS.

 

Using It

  • There's a folder called www in your W drive or PASS account.
    • Put all the files you want to have on your web site within this folder.
    • Everything within this folder is visibile to anyone on the Internet.
    • You can make folders within this folder that are also visible.
  • Your web page URL will be of the form http://www.personal.psu.edu/abc123/.
    • Everything that's in the www folder is directly accessible at this location.
    • Ex: If you have a file called schedule.htm in the www folder, its URL is http://www.personal.psu.edu/abc123/schedule.htm.
    • If no filename after slash, the default file index.html or index.htm is displayed. (Every folder should have one!)

In Windows, use the drag-and-drop methods to put your files in W:\www. In PASS, use the appropriate buttons.

 

Keeping web servers happy

  • Never use spaces in filenames; separate words with underscores instead
  • Never use capital letters in filenames
  • Never use special characters in filenames

Basic file naming convention: letter followed by one or more letters, numbers, or underscores followed by a dot followed by an extension
