Rowspan and Colspan

The <td> tag has two attributes, rowspan and colspan, that allow you to make cells cross multiple rows and columns. Essentially, the idea is that cells could be merged together into one.

Both attributes take the number of rows or columns to span as the argument.

It's important to note that when a cell spans multiple columns, there are fewer cells in that row than in other rows. HTML simply fills in cells in a row until the row is closed. If there are too many cells defined in a row, they still show up, but the table takes on a deformed look. I recommend that you try putting too many cells in a row when you're first experimenting with colspan in lab to see what this looks like (so you know how to spot an error to debug it).

Here are some examples:

 

HTML As displayed in browser 
<table border="1">
  <tr>
    <td colspan="2">first</td>
    <td>second</td>
  </tr>
  <tr>
    <td>third</td>
    <td>fourth</td>
  </tr>
</table>
