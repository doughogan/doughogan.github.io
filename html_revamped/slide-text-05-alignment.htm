Alignment

We learned about the <p> tag to create paragraphs. Alignment is a property of a paragraph, so it is set as an attribute. The attribute name is align. It can take on one of four values:

  • left
  • center
  • right
  • justify

Example

HTML As displayed in browser 
This part isn't formatted and longer than the rest.
 This part isn't formatted and is longer than the rest. 
<p align="center">This part is centered.</p>

This part is centered.

 
<p align="right">This part is right-aligned...</p>
<p align="center">...but this part is centered.</p>

This part is right-aligned...

...but this part is centered.

 
<p align="right">
   <font color="#336600">This part is green and right-aligned.</font>
</p>

This part is green and right-aligned.

 

Notes

  • The default alignment is left. If the align attribute is omitted, text is left-aligned.
  • The justify alignment is a newer addition to the language and is not supported by all browsers. Do some trials to see which ones do support it.
  • Note the nesting of tags in the last example.
