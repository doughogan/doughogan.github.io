How do we write it?

HTML files are plain text files, so a simple plain text editor will work fine. The simple one in Windows is Notepad. The simple one in Mac OS X is TextEdit.

Notepad

It's usually found under Start > Accessories > Notepad.

Very limited set of commands, but if you can use Office, you can certainly learn Notepad.

When you save in Notepad, the default file type is txt. For HTML, you want the extension html or htm. To achieve this, type the filename in double quotes in the Save As dialog, e.g. "index.html"

Text Edit

Double-click the Macintosh HD icon and go to the Applications folder. Look for TextEdit.

I'd recommended right-clicking the TextEdit icon in the dock and choosing Keep In Dock so you don't have to hunt it down every time.

We want plain text, so we need to go to Format > Make Plain Text to achieve this.

Some first-time setup to make life much easier:

  1. Go to the TextEdit menu > Preferences.
  2. While New Document is selected, change the Format to Plain Text.
  3. Go to the Open and Save section of the Preferences. Here,
    • Uncheck Add ".txt" extension to plain text files.
    • Uncheck Ignore rich text commands in HTML files.
  4. Close the Preferences window.

Now, saving HTML files will always be easy. For HTML, you want the extension html or htm, so just type the filename in the Save As dialog, e.g. index.html.
