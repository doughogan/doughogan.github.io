Anchors

An anchor is a point within an HTML file. They are useful if we wish to make a link to specific part of a page, rather than the whole page.

Anchors require a form of the <a> tag we saw earlier to define them:

<a name="name_of_anchor"></a>

Links can be made to anchors within the current page by using the standard link tag with a path beginning with a #:

<a href="#name_of_anchor"></a>

Links can be made to anchors in other pages by appending a a # and an anchor name to the the path:

<a href="filename.html#name_of_anchor"></a>

Lots of example uses of anchors can be found throughout the class notes.
