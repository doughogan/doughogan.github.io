CS 100 Lab 9: Python 2

The goal of this lab is to put into practice ideas from last Friday's lecture. There were two main ideas: short IPO standalone programs and loops. You'll complete two small programs, one for each idea.

Part I: Compound Interest

Recall the compound interest formula, which tells how much an investment is worth after interest has accumulated after a given number of periods:

An = P(1+i)n

where:

  • P is the principal, or amount of money initially invested
  • i is the interest rate per payment period as a fraction (0 ≤ i ≤ 1; 5% interest would be treated as 0.05 here)
  • n is the number of payment periods
  • An is the amount the investment is worth after interest has accrued on the given principal at the given interest rate after n payment periods

Your job is to write a short program that tells a user how much money he/she has in a savings account that earns interest monthly, after making an initial deposit, never withdrawing anything, and letting the interest grow. Use the IPO model. In other words:

  1. Have an input phase where you prompt the user for how much money was deposited, what the interest rate is, and how many months have passed. Remember to make your prompts clear. Store these values in three variables.
  2. Compute the value of the investment, using the formula above.
  3. Report your result to the user. Remember to label the units.

 

Part II: Finding a Minimum

Write a short Python program that asks a user to input as many postive numbers as desired, using -1 to stop input. Report the minimum value entered.

Hint: Look at all the example code snippets. This is meant to combine the ideas from a few of them, with slight modifications.

For check-plus, after you've achieved everything else, also tell how many values were entered and how many of them were multiples of 10.
