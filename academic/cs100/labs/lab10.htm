Lab 10: Starting with Databases

Last Modified 3/1/16

This lab is just about getting started with Access and preparing some database tables you'll use in future labs. It follows from the class demo from Friday and experience from the class activity with the courses database a while back.

Here are your tasks:

  1. Launch Access and create a new Blank Desktop Database as per the demo in class.
  2. Create a database table called courses and set it up to contain the same fields as the one we did in class. Do all of the setup in Design View, providing descriptions for each of the fields (but you can copy mine).
  3. Switch over to Datasheet View and populate the courses table with information on 5 different courses.
  4. Go back to Design View and create a new table called books. It should allow us to specify the following about a book:
    • an ID (what's special about this?)
    • title
    • author
    • whether the book is paperback or hardcover
    • how many pages are in the book
    • what year the book was published
    • the book's publisher
    • how much the book costs
    • the book's ISBN
  5. Switch back to Datasheet View and add some book information. You must have information about 15 different books. Start with 5 real books, then either make up whatever you want for the rest or base them on slight changes to the existing books. For example, you might make a paperback edition of a hardcover book that costs less or an abridged edition that costs less and has fewer pages. Follow advice from the TA on any suggestions about your data.
  6. Recall our demo on how to get to query view from Friday. Go to the SQL View and run the following query:
       select * from books.
