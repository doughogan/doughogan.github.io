Lab 13: Word Processing

Last Modified 10/31/16

Use what you learned from the tutorials to modify a document.

  1. Download and open http://www.cs.uic.edu/~hogand/cs100/lab13document.doc.
  2. Change the font size to be size 10 points.
  3. Change the font face to be Times New Roman.
  4. Change the headings for different sections in the document to look like headings. (Think of what we did in HTML, except now you must make Word do something similar.)
  5. Make the margins be one inch.
  6. Add at least 3 footnotes.
  7. Add at least 3 entries in the bibliography.
  8. Add a title page
  9. Add page numbering on all pages except the title page.
  10. Make a table of contents that looks like the following:
    Introduction ..................................
    History of Computing ..................
    Components .................................
    Control Unit .................................
    Arithmetic Logic Unit (ALU) .....
    Memory .......................................
    Input/Output (I/O) .......................
    Networking and the Internet ........
    Programming Languages .............
    Low-level languages ....................
    Higher-level languages ................
    Misconceptions ............................
    Required Technology ....................
