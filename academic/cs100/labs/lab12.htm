Lab 12: SQL Queries with Two Tables

Last Modified 3/1/16

This lab builds on your last two labs. Here are your tasks:

  1. Edit your table of courses so that it also has a field for the course textbook. Use a foreign key to the books table to achieve this.
  2. Edit the data in your courses table to make all of the courses have an associated textbook, even if it's not really the book.
  3. Write a query using a subquery to display the full course numbers of all courses, along with the title, author, and cost of their textbooks.
  4. Write a second query that achieves the same objective as #3, but this time using an inner join.
  5. Now edit either query so that you only display 100-level courses and they are listed with the course with the most expensive book first, going down to the cheapest book.

Like you did in the last lab, write down your queries for #3-#5 on paper and hand that in to the TA at the end.
