The <th> Tag

In all of these examples, we've worked with building tables from <tr> and <td> tags. Of course, we need to format header rows and columns in bold or italic print to distinguish them from the actual data cells. We could do this by nesting formatting tags within the cells.

We started this way to keep things simple. There is also a <th> tag, which stands for "table header." We can use <th> in lieu of <td> when we want to mark cells as part of a header row or column. By default, the contents of these cells are formatted with bold text and centered, so applying the formatting is much simpler.

Consider the following two examples:

HTML As displayed in browser 
<table cellpadding="5" cellspacing="1" border="1">
    <tr>
       <td>&nbsp;</td>
       <td align="center"><strong> column 1 header </strong></td>
       <td  align="center"><strong>column 2 header </strong></td>
    </tr>

    <tr>
       <td align="center"><strong>row 1 header </strong></td>
       <td>first</td>
       <td>second</td>
    </tr>

    <tr>
       <td  align="center"><strong>row 2 header </strong></td>
       <td>third</td>
       <td>fourth</td>
   </tr>
</table>
