Fonts

The <font> tag needs attributes to be useful. Specifically, it can take any of these attributes:

  • size - takes arguments of specific HTML font sizes 1-7 (usu. bad) or relative font sizes like "+1" or "-1" (good)
  • color - takes arguments of hexadecimal codes for colors (good, but you have to look them up) or color names (okay if they're known by the browser)
  • face - takes arguments of font faces (dangerous, assumes all users have the font installed) or comma-separated lists of font faces (better)

Example

HTML As displayed in browser 
This part isn't formatted.
 This part isn't formatted. 
<font size="+1">This part is a bit bigger.</font>
 This part is a bit bigger. 
<font size="-1">This part is a bit smaller.</font>
 This part is a bit smaller. 
<font color="#336600">This part is green.</font>
 This part is green. 
<font face="Arial, Helvetica, sans-serif">
  This part is in Arial, Helvetica, or sans-serif.
</font>
 This part is in Arial, Helvetica, or sans-serif. 
<font color="#336600" size="+2">
  This part is green and a bit bigger.
</font>
 This part is green and a bit bigger. 
<font color="#000099" size="1">
  This part is blue and in size 1 and <b>this is bold</b>.
</font>
 This part is blue and in size 1 and this is bold.
