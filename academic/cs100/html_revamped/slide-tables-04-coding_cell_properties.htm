Attributes of <table> for Whole Table

Again, we set attributes of the <table> tag:

  • align
  • width

Valid options:

  • align is set to left by default. We can choose center or right. Remember, this is the alignment of the whole table on the page.
  • width can either take a number of pixels or a percentage. Note that the percentage tells what percentage of the width of the browser the table takes up, or, if the table is nested within something else, say another table cell, the percentage of the parent element.

Example

HTML As displayed in browser 
<table cellpadding="5" cellspacing="1" 
       border="1" align="center">
  <tr>
    <td>first</td>
    <td>second</td>
  </tr>
  <tr>
    <td>third</td>
    <td>fourth</td>
  </tr>
</table>
first second
third fourth
