Tags With Attributes

With the exception of the <a> tag, all of the tags we've worked with have simply contained the name of the tag between the angle brackets. We now add a slight complication into the mix...

Some tags can have attributes that can be set to specify certain properties. The attributes are set in the opening tag. The general form of the syntax is:

<tag-name attribute_1="argument" attribute_2="argument" ... attribute_n="argument">

Note that the closing tag does not include the attributes. It just needs the slash and the name of the tag:

</tag-name>
