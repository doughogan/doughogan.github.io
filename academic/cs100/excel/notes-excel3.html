Absolute Referencing

vs. Relative Referencing

Recall: What happens to cell references when you copy a formula to a different cell?

That's called relative referencing because the cell references change relative to where the formula is.

Absolute Referencing

There's a contrast to relative referencing called absolute referencing. Simply put, absolute referencing means that part or all of the cell references remain "locked" when the formula is changed.

You can lock a row reference, a column reference, or both. To lock a reference, put a $ in front of the row or column reference.

 

Examples of Absolute Referencing

For a reference to the cell C3, the three degrees of absolute referencing:

What is locked Reference
row
C$3
column
$C3
cell
(i.e. row AND column)
$C$3

Absolute referencing can be used in ranges too, e.g. $A$1:$D$5.

Conditional Functions: The IF function

The IF function implements the notion of selection. Here is the syntax:

=IF(condition, value_if_true, value_if_false)

The three parameters to the function:

  • condition -- For the argument, give an expression that is a logical test that returns either TRUE or FALSE. Use the Boolean operators below to build your expression.
  • value_if_true -- For the argument, tell Excel what to put in this cell if the condition is TRUE. Give an expression that could be any of the following:
    • A constant (a number)
    • An arithmetic expression built up from constants or cell references
    • A function call
    • An arithmetic expression built up with function calls
    • A text string enclosed in quotes (or "" for leaving the cell blank)
  • value_if_false -- For the argument, tell Excel what to put in this cell if the condition is FALSE. Give an expression as above.

We'd like a formula for B6 which tells whether on not the GPA in A6 is a Dean's List GPA.

We begin with a conditional function framework:

=IF(condition, value_if_true, value_if_false)

We want to test whether the GPA is at least 3.6. The GPA is found in A6, and we want at GPA of at least 3.6. We can use the logical test A6>=3.6 for this case. The formula looks like this:

=IF(A6>=3.6, value_if_true, value_if_false)

Now, we need to specify what to do if the test is true. We want the cell to have a message saying the student made the Dean's list, so we'll use a text string enclosed in quotes for the value_if_true:

=IF(A6>=3.6, "Made Dean's List" , value_if_false)

If the test fails, we need another message. We'll use a simple one:

=IF(A6>=3.6, "Made Dean's List" , "No" )

The above formula will be appropriate. It could be improved. How?

Conditional Functions: The VLOOKUP function

You can implement most multiple selection problems using nested IF statements. That becomes tedious, especially balancing all of the parentheses. A more sensible abstraction is the VLOOKUP function.

Lookup Table

VLOOKUP requires you to create a matrix first with cutoffs for the ranges you're looking up and what the result will be for each given range.

The first column of the lookup table must be sorted in ascending order. If your VLOOKUP function isn't working, this is likely the problem.

We'll begin by creating a sample table of a mapping of numeric grades into letter grades:

 
A
B
C
D
1 Cutoff Letter Grade    
2 0 F    
3 60 D    
4 70 C    
5 80 B    
6 90 A    
7        
8        

Problem

What is the address of this table?

 

VLOOKUP Syntax

Syntax:

=VLOOKUP(cellToLookupInFirstRowOfMatrix, addressOfMatrix, columnOfResult)

The three parameters to the function:

  • cellToLookupInFirstRowOfMatrix -- For the argument, give the address of the cell you're looking up in the table.
  • addressOfMatrix -- For the argument, give the cell address of the lookup matrix. In almost all cases, you'll want to use absolute referencing.
  • columnOfResult -- For the argument, give a natural number representing the column whose value you want to display as the result. The first column in the matrix is "1"; the second is "2"; etc. Usually, you'll use "2" here, but if you have a larger table, larger numbers maybe appropriate.

 

Letter Grade Example

Suppose we're in the same sheet as before and have the following data:

 
A
B
C
D
11 Number Grade Letter Grade    
12 95      
13 76      
14 82      

Write a formula for B12 to look up the letter grade.

Three arguments:

  • cellToLookupInFirstRowOfMatrix = A12 since that's where the number grade is.
  • addressOfMatrix = $A$2:$B$6, from above.
  • columnOfResult = 2, as we want letter grades from the 2nd column of the table.

The result:

=VLOOKUP(A12, $A$2:$B$6, 2)

 

Exact match or Range lookup?

There may be times when you want to force Excel to find an exact match in your lookup table. To do so, add a fourth argument of FALSE. If it's omitted, Excel assumes TRUE, that you do want to find the closest matching range.
