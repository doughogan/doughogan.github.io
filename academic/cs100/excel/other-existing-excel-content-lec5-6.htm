Working with Multiple Worksheets

Worksheet = one spreadsheet, what we've been using all along

Several worksheets make up a workbook, which is what is really stored in an Excel file.

Why might one use mutliple worksheets?

You can find the worksheets in a workbook by looking at the sheet tabs in the bottom left of the Excel window.

Worksheet Tasks

Task How to do it
view a worksheet
  1. click on its sheet tab
rename a worksheet
  1. double-click its sheet tab
  2. type a new name
  3. press Enter

  1. right-click on the tab for the sheet after where you'd like a new sheet

  4. double-click a location (you move the sheet to be before your choice)

Addressing Across Multiple Worksheets

Addressing formats

The sheet name can be added to any cell address. It's separated from the cell address by an exclamation point. (If no sheet name is given, Excel assumes the current sheet.)

  • If the sheet name does not include spaces: SheetName!CellAddress
  • If the sheet name does include spaces: 'Sheet Name'!CellAddress

You can use the mouse here too: whenever you want to use a cell address from another sheet, click its sheet tab first, then select the cell(s), then continue editing your formula in the formula bar.

 

Examples

  • =Sheet2!A1+Sheet4!A1
  • =SUM('New Name for this Worksheet'!A2:A5)

Sorting Data in Excel

Motivating Example

Let's look at a slightly modified version of the GPA spreadsheet.

 

Steps to Sort

  1. Highlight the appropriate part of the sheet (as per the example)
  2. Go to Data > Sort
  3. Choose options for sorting in the Sort dialog:
    • Sort by: Here you choose the column by which your data will be sorted.
      • All other highlighted data in the same row will travel along.
      • The new order of the rows will be based on this column.
      • If you have headers entered in your worksheet, they will appear here, rather than "Column A," "Column B," etc.
    • Sort Order: You can sort in one of two orders:
      • Ascending (alphabetical and counting …1, 2, 3,…)
      • Descending (reverse alphabetical and counting …3, 2, 1…)
    • Header Row?
      • If the first row you highlighted contains titles for each of the columns, choose the Header Row option. This first row will then be ignored in the sorting, and will remain the first row.
      • Otherwise, choose No Header Row.
  4. Click OK.

 

"Tie-Breaking"

If two values in the column you choose to Sort by are the same, you may want a method of resolving "ties." Excel lets you do three levels of sorting.

In the Sort dialog from step 3 above, use the Then by fields for multiple levels of sorting.
