Range Addressing

It will be useful to have a way of referring to a range of cells.

  • Contiguous ranges: the colon is used to denote a contiguous range, inclusively. Give the address of the upper left cell of the range, then a colon, then the lower right cell of the range.
  • Non-contiguous ranges are separated by commas (and possibly spaces, but they're not necessary).

Abstraction Revisited and Excel Functions

Recall: Abstraction

  • Focus on the WHAT, not the HOW
  • Allows us to use complicated things without understanding how they work

Function Warm-Up

The notion of abstraction applies to functions in Excel.

In your GPA assignment, you should have had a formula that looked like this:

=E7+E8+E9+E10+E11

We could use a function called sum to achieve the same task:

=SUM(E7:E11)

Why does abstraction apply here?

 

Function Syntax and Terminology

A function is an abstraction of some complex calculation. Technically, we're doing a function call. The syntax for a function call, generally, is:

=function_name(argument_list)

Things in blue are required for every function call. Things in black are replaced appropriately.

What goes in the parentheses in called a list of arguments to the function. The operation for the sum function is exactly the same whether it is given the argument range E7:E11 or the argument range A1:A2.

 

Discussion: A Word About Syntax

Consider the following two formulas:

  • =A1 + A2
  • =SUM(A1, A2)

Some Excel Functions

Function Sample Use Description/Return Value
=SUM (range) =SUM(A1:A5)
=SUM(A1:A5,B5) 		Returns the sum of the cells within the given range
=AVERAGE(range) =AVERAGE(B4:F4) Returns the average of the cells within the given range
=COUNT (range) =COUNT(A1:A100) Counts how many cells in the argument have numerical values, returns the count
=MIN (range) =MIN(B2:B50) Returns the smallest value in the range
=MAX (range) =MAX(B2:B50) Returns the largest value in the range
=LOG (x, base) =LOG(B2,10) Returns the logarithm (of the specified base) of x
=EXP (n) =EXP(2) Returns en, or Euler's number raised to the power of n
=PI () =PI() Returns π, the mathematical constant pi (stored in the computer's memory to several decimal places)
=RAND () =RAND() Returns a random number between 0 and 1 (inclusive of 0 but exclusive of 1)
=SIN(angle) =SIN(PI()/2) Returns the sine of the angle (measured in radians), likewise for COS and TAN
=ASIN(number) =SIN(A2) Returns, in radians, the inverse sine (arcsine) of the number, likewise for ACOS and ATAN
=RANK (cell, contiguous_range) =RANK(A1,A1:A15) Returns the rank of the cell within the given contiguous range.

Nested Function Calls

You can use the results of one function call as an argument to another function call.

The "=" is not repeated with the nested calls; it only appears at the beginning of the formula (unless it's used to test for equality).

Examples

  • =SIN(F8*PI()/180)
  • =ASIN((SIN(F8*PI()/180)*C8)/E8)*180/PI()

Graphs from Ordered Pairs

Often, when need to make graphs, you are given data in the form of ordered pairs and you want to plot those points.

Steps:

  1. Have all of the x-axis values in one column.
  2. Have all of the y-axis values in another column (you could have multiple y-axis columns if you want to plot several curves on the same graph)
  3. Select the x-axis column and the y-axis column(s).
  4. Go to the Chart Wizard.
  5. Select XY (Scatter) as the Chart Type and an appropriate way of "connecting the dots."
  6. Continue with the chart wizard.

Caution: do not select the y-axis column labels! This will distort the scale.
