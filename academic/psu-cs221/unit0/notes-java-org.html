History of Java

(This material comes from Section 1.9 of Deitel.)

  • Designed to be based upon C++ (in turn, based upon C, and in turn BCPL and B).
  • Created in 1991 by James Gosling.
  • Initially called Oak after a tree.
  • Renamed Java during a conversation at a coffee shop after it had been discovered that Oak was already a language.
  • Formally announced at an industry conference in 1995.

The Organization of Java

Recall Question: What's the difference between an compiled language and an interpreted language?

 

Java is unique in that it does not fall distinctly into one of these categories. Instead,

  1. We compile source code (a .java file) into an intermediate state called bytecodes (a .class file).
  2. Bytecodes are run by the Java Virtual Machine (JVM). There are various flavors of JVMs for various operating systems.

The beauty of this is that it allows platform independence. A Java program can be compiled on any operating system and run on any JVM. Naturally, this makes the language web-friendly.

In running a program, the JVM does the following:

  1. Loads the program into memory via the class loader. Any .class files the program uses are loaded.
  2. Calls upon the bytecode verifier to make sure there are no problems (or security issues).
  3. Executes the bytecodes. This is essentially interpretation using a tool called the Just-In-Time compiler.
