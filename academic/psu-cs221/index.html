CMPSC 221 Notes

I'm posting here lecture notes I developed in 2009 and maintained over the Fall 2009 to Spring 2012 for the CMPSC 221 course at Penn State. This is a course in object-oriented programming, graphics, event-driven programming, GUIs, and relational database basics in SQL and JDBC. The target language is Java. This course is/was the third in the introductory programming sequence. The first two were in C++; this course begins with a transition to Java given programming background in C++ (with a bias toward C).

To be sure, some things have changed in the last decade. Applets have become deprecated; arguably they were sluggish back then. I could imagine converting everything to applications and JFrames, but my goal is to share these resources for self-education. Maybe someday.

The course has three major units and a short introductory unit transitioning to Java:

  • Unit 0: Transition to Java from C++
  • Unit 1: Object-Oriented Programming
  • Unit 2: Graphics, Event-Driven Programming, and GUIs
  • Unit 3: Basics of Relational Databases, SQL, and JDBC

I used the Deitel and Deitel Java: How to Program series as an optional textbook along the way. Any textbook references refer to the 10th edition of that book.
