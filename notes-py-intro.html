Background and Getting Started

We turn now to Python, which is a general-purpose programming language in its own right. We can use Python to do very simple calculations, but we can also use Python to build very robust, large software. The language is among the top programming languages used today. In this course, we'll look at some of its features, really focusing on ideas that are common to many languages. In a short time, both because of the nature of Python and because of what we've already covered in this course, we'll be able to gain exposure to lots of important programming concepts.

Recall when we talked about HTML needing an interpreter, i.e. a program that can read and execute HTML code (the browser in that case). The same is true for Python. But, more generally speaking about programming languages, it falls into the second of these two categories:

  • compiled languages: those where programmers write code and run it through a program that checks its syntax and generates an executable program
  • interpreted languages: those where no advance syntax check is done, but instead the language needs an interpreter to run code; errors are handled on the fly (as best as the system can)

Software for working in Python is available for free if you want to install it on your own machine. (It's often pre-installed on Mac OS X these days.) It's installed in our lab already, so you don't need to do anything to meet your course requirements. These are various in-browser Python intepreters you can use too.

Modes and Starting the Interpreter

There are two (for our purposes, anyhow) different ways we can think of using Python:

  • In interactive mode, where we use Python sort of like a calculator. Once we start Python interactive mode, we can give it commands and it will respond. This is how we'll start today, to get a feel for the language.
  • By writing code in a separate file and running that file through the Python interpreter. This is more like what we did with HTML and how we would write software we intend to run many times.

So then, in either case, we'll be operating in a command prompt environment. For now, to start the Python interpeter...

  1. First, get to a command prompt environment:
    • On Mac OS, go to Macintosh HD, look for Applications, then Utilities, then open Terminal. Or, search for Terminal in Spotlight. (I recommend right-clicking it in the Dock and selecting the Keep in Dock option.)
    • On Windows, search for Command Prompt in the program search. (Or go to Start > Run and enter "cmd.")
  2. Type the command python and hit enter.

After this, you'll see some information, followed by >>>. It is after the >>> prompt that you enter commands and then press "enter." That produces a result and another >>> prompt where you may enter additional commands.

Variables, Take 1

As in algebra, we can use variables to represent quantities abstractly, whether we know their values or we don't. Variables can hold different types of data, but today, we're using Python like a calculator, so let's work just with variables holding numbers.

The key thing you need to know now is how variable names look. As will generalize to other programming languages as well,

  • Variable names start with letters
  • Variable names may NOT contain spaces
  • Variable names may include more letters or digits or underscores
  • Variable names may NOT contain any special characters other than underscores

Here are some examples:

Valid Not Valid

a
a2
length
age
myVariable
numberOfStudents
number_of_students

2
a 2
length/count
%OfEarnings
2nd_value
number of students

Convention is that variables names start with lowercase letters and either use capital letters or underscores to indicate new "words" within, as you see in the last three valid examples.

Operators

There are three kinds of operators that are important. You encountered two of these three groups in Excel, and for comparison, I'll include the Excel equivalents too. We'll look at each of the groups.

Arithmetic

The (binary) arithmetic operators are those which take two numeric inputs and compute an output. The result is another number. Here are the arithmetic operators:

Python Operator Excel Operator Python Example Arithmetic Expression Meaning
+ + a + b addition
- - curYear - yearBorn subtraction
* * length*width multiplication
/ / total/count division
** ^ base**exponent exponentiation/power
% N/A
(use a function)		 changeDue%25 modulus/remainder

(Note that for division, // is also defined. This is for dividing integers by integers and wanting a whole-number quotient.)

 

Relational Operators (or Boolean Operators)

The relational operators are used to compare two numeric inputs. The result is either True or False. As with Excel, we'll use these in forming the conditions for if statements. Here are the relational operators:

Python Operator Excel Operator Python Example Boolean Expression Meaning
> > a > 5 greater than
>= >= reading >= threshold greater than or equal to
< < input < 0 less than
<= <= input <= 0 less than or equal to
== = value == 33 equal to
!= <> input != 1 not equal to

Pay special attention to equality. A single = means something different.

 

Assignment

The assignment operator is new to you. It is =. It is used as follows:

<variable name> = <arithmetic expression>

This is a rather important idea, so, on to the next section...

Variables, Take 2, and Assignment

Let's combine the ideas of interactive mode Python, arithmetic operators, and variables now.

First, you can simply enter arithmetic expressions after prompts to compute the results, just like using a calculator. Here are some examples:

>>> 1 + 2
3
>>> 300 - 3
297
>>> 92%5
2
>>> 2**5
32

This works, but the results are forgotten as soon as we move on to the next calculation. We'd like to use variables and the assignment operator to store values. Here's how it works:

  1. Type the name you want to use for a variable (exactly one variable name)
  2. Type the assignment operator =
  3. Type the value you want to store in the variable or an arithmetic expression to compute it

Here a few examples:

>>> a = 5
>>> b = 1 + 2
>>> c = 2**5

What happens is that the values are stored, rather than being displayed. As long as we don't close the terminal window, they will be remembered. To recall the value, of a variable, simply type its name at a prompt. So, we could recall the value of those three variables we defined:

>>> a
5
>>> b
3
>>> c
32

A lot of power lies in the fact that we can use values we've computed in other calculations. Continuing from above, we could do as follows:

>>> d = c + 1
>>> d
33
>>> b = b*10
>>> b
30
>>> b = b*10
>>> b
300

Notice that the system is substituting the last stored value of a variable on the right hand side of the assignment operator, so we can write things like b = b*10. It is important to read the assignment operator as "gets the value" rather than "equals." Something like b = b*10 would make no sense if it were an algebraic equation, but assignment statements are not equations!

Exercise: What is the value of the variable after each step of the following:

>>> count = 0
>>> count = count + 1
>>> count = count + 1
>>> sum = 0
>>> sum = sum + 7
>>> sum = sum + 3
>>> average = sum/count

 

 

One More Example: In our first Excel lecture, we derived the following formula that we put in cell C4:

=3.14159*(B4^2)*A4

Note that in Excel, we didn't say where we were storing the results of calculations, because that was always the cell where we wrote the formula. In some sense, we were storing the result in C4, so perhaps we could think of the above like:

C4 = 3.14159*(B4^2)*A4

Translating to Python and giving some values to similarly-named variables:

>>> b4 = 2
>>> a4 = 10
>>> c4 = 3.14159*(b4**2)*a4

But, even better, we can use more descriptive names in Python:

>>> pi = 3.14159
>>> radius = 2
>>> height = 10
>>> volume = 3.14159*(radius**2)*height

Output (Print) Statements

As we work toward the idea of using Python as a general purpose programming language and not just a calculator, we need the idea of output, or displaying information to the user. Perhaps that information is a message (all text). Perhaps it's the result of a calculation. Perhaps it's both, such as a labeled result of a calculation.

In any case, the simplest way to do this is via the print function. Calling a function in Python is similar to calling a function in Excel. Namely, you must provide the name of the function, then give it an argument or arguments in parentheses afterward. For our purposes, these are the kinds of inputs the print function can take:

  • Text in quotes (called a string)
  • A variable name, where what is displayed is the name of that variable
  • Any number of the above with commas between.

Here's an example interaction that just displays text:

>>>> print("hello")
hello

Here's another example that might follow the volume calculation from earlier:

>>> print("Radius in m: ", radius)
Radius in m:  2
>>> print("Height in m: ", height)
Height in m:  10
>>> print("Volume in cubic meters: ", volume)
Volume in cubic meters:  125.6636

If Statements

Recall how we studied the if function in Excel last week:

=IF(condition, value_if_true, value_if_false)

This is one realization of the important idea of making decisions in programming. In most programming languages, though, instead of thinking of a value we compute when the condition is true or false, we simply have statements. So far, in Python, that means either assignment statements or print statements (calls to the print function).

Here's the general form of the equivalent construct in Python:

if condition:
   statement(s) to execute if true
else:
   statement(s) to execute if false

Recall our Excel example:

=IF(A6>=3.6, "Made Dean's List", "No")

We might implement this in Python as follows:

if a6 >= 3.6:
   print("Made Dean's List")
else:
   print("No")

Or, better:

if gpa >= 3.6:
   print("You made the Dean's List")
else:
   print("Sorry, you didn't make the Dean's List")

Here's another example: 

if a % 2 == 0:
   b = a**2
else:
   b = a + 1

Here's yet another example: 

if hoursWorked >= 40:
   overtimeHours = hoursWorked - 40
   overtimePay = hourlyWage*overtimeHours*1.5
   pay = hourlyWage*hoursWorked + overtimePay
else:
   pay = hourlyWage*hoursWorked

Notice two important things in all of these examples:

  • There is a colon at the end of the if and else lines.
  • The statement(s) executed in each case must be indented. I recommend three or four spaces. Whichever you choose, you must be consistent.

Because an if statement uses multiple lines, you'll notice that in the interactive mode, you must hit "enter" twice after entering the last line.

Finally, note that it's entirely possible to omit the else clause entirely, e.g. : 

if gpa >= 3.6:
   print("Congratulations, you made the Dean's List")

There's much more we can do with this idea, but for now, the above is all you need to know. Note that there is also the idea of one or more "else if" clauses, using the keyword elif. The idea of nesting if/else blocks like we did with function calls in the Excel example works in Python too.

Simple Loops

As a final idea in our introduction to Python, consider the notion that you may want to do the same thing (or a similar thing) several times in a row. This is the notion of repetition in programming, and, certainly, you've seen several applications of it informally in our conceptual discussions. Repetition is something we can finally realize via code, and we do that via something called a loop.

The simplest kind of loop is called a for loop, and we'll look at in the case of counting. We need some variable called a loop control variable or LCV, and it really functions as a counter. Here's a general form:

for i in range(start_value, one_after_stop_value, step_value):
   statement(s) to be repeated

If you simply want to do something n times, you could do the following:

for i in range(0, n, 1):
   statement(s) to be repeated

Here's an example of printing the same message 5 times: 

for i in range(0, 5, 1):
   print("hello")

Here it is 10 times:

for i in range(0, 10, 1):
   print("hello")

But, the power lies in leveraging the LCV. Here's how we can count from 1 to 10:

for i in range(1, 11, 1):
   print(i)

Or maybe we could print out the first several odd numbers:

for i in range(1, 30, 2):
   print(i)

Or we could do a calculation like in this example:

for i in range(1, 11, 1):
   print(i, 2**i)

That's a nice start.

Notice, again, the indentation and the color at the end of the for line. These are required, just as with if statements.
